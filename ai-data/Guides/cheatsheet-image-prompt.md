# Cheatsheet Image Design Prompt

**Use this prompt with design tools (Canva, Figma) or AI image generators to create your LinkedIn cheatsheet graphic.**

---

## For Canva/Figma (Manual Design)

### Step-by-Step Instructions

1. **Create Canvas**
   - Size: 1200 x 627 pixels (LinkedIn post size)
   - Or: 1080 x 1080 pixels (square format)

2. **Left Panel Setup (40% width)**
   - Add your professional portrait photo
   - Position: Center-left
   - Apply subtle blur to background if needed
   - Add tech logos around photo:
     * Top-left: [Logo 1]
     * Top-right: [Logo 2]
     * Bottom-left: [Logo 3]
     * Bottom-right: [Logo 4]
   - Logo opacity: 70-80%

3. **Add Title Text (Left Panel)**
   - "CHEATSHEET" - Large, bold, yellow (#FFD700 or similar)
   - Font size: 72-96pt
   - Position: Top of left panel
   - "to crack" - Medium, white (#FFFFFF)
   - Font size: 36-48pt
   - Position: Middle
   - "[YOUR TOPIC] Interviews" - Large, bold, yellow
   - Font size: 60-72pt
   - Position: Bottom of left panel

4. **Add Arrow**
   - Yellow arrow pointing right
   - Position: Bottom-right of left panel

5. **Right Panel Setup (60% width)**
   - White or light gray background (#F5F5F5)
   - Add numbered sections (1-9 or as needed)
   - Each section format:
     ```
     1. [Section Title]
        Tools: [Tool 1], [Tool 2], [Tool 3]
        Concepts: [Concept 1], [Concept 2]
        Purpose: [One-line description]
     ```

6. **Typography**
   - Section numbers: Bold, 32-40pt, dark blue (#1E3A8A)
   - Section titles: Bold, 24-28pt, dark blue
   - Tools/Concepts: Regular, 16-18pt, dark gray (#4B5563)
   - Purpose: Regular, 14-16pt, dark gray
   - Font: Sans-serif (Montserrat, Poppins, Inter, or similar)

7. **Spacing**
   - 40-60px padding around all edges
   - 30-40px spacing between sections
   - Consistent left alignment

8. **Final Touches**
   - Add subtle shadows to text for depth
   - Ensure high contrast for readability
   - Check all text is readable at small sizes
   - Export as PNG (high quality, 300 DPI)

## For ChatGPT/DALL-E (Text-to-Image)

### Detailed Prompt

```
Create a professional LinkedIn post graphic with these exact specifications:

DIMENSIONS: 1200 pixels wide x 627 pixels tall

LAYOUT: Split-screen design, left panel 40% width, right panel 60% width

LEFT PANEL ELEMENTS:
- Professional headshot portrait in center-left position
- Four technology logos positioned around the portrait: one top-left, one top-right, one bottom-left, one bottom-right (logos should be semi-transparent, 70% opacity)
- Large bold yellow text at the top saying "CHEATSHEET" (font size approximately 80pt)
- Medium white text in the middle saying "to crack" (font size approximately 40pt)
- Large bold yellow text at the bottom saying "[YOUR TOPIC] Interviews" (font size approximately 65pt)
- Yellow arrow icon pointing horizontally to the right, positioned at bottom-right of left panel

RIGHT PANEL ELEMENTS:
- Clean white or very light gray background (#F5F5F5)
- Nine numbered sections, each with:
  * Bold dark blue section number and title (font size 28pt for title)
  * "Tools:" label followed by comma-separated tool names in dark gray (font size 16pt)
  * "Concepts:" label followed by comma-separated concepts in dark gray (font size 16pt)
  * "Purpose:" label followed by one-line description in dark gray (font size 14pt)
- Generous spacing between sections (approximately 35px)
- Consistent left alignment
- Professional sans-serif font (like Montserrat or Poppins)

COLOR SCHEME:
- Primary: Yellow/Gold (#FFD700) for titles and arrow
- Text: Dark blue (#1E3A8A) for section titles, dark gray (#4B5563) for body text
- Background: White (#FFFFFF) or light gray (#F5F5F5)
- High contrast for readability

STYLE: Modern, professional, corporate, clean, infographic style, suitable for LinkedIn, high quality, sharp text, no blur
```

## Export Settings

### For LinkedIn:
- **Format:** PNG or JPG
- **Quality:** High (90-100%)
- **Resolution:** At least 1200px width
- **DPI:** 72-150 (web quality)
- **File Size:** Under 5MB (LinkedIn limit)

### For Print (if needed later):
- **Format:** PNG
- **Resolution:** 300 DPI
- **Color Mode:** RGB

---

## Common Mistakes to Avoid

1. ❌ Text too small (won't be readable on mobile)
2. ❌ Too many colors (keep it simple)
3. ❌ Low resolution images (blurry logos/photos)
4. ❌ Poor contrast (text hard to read)
5. ❌ Too much text (overwhelming)
6. ❌ Inconsistent spacing (looks unprofessional)
7. ❌ Wrong dimensions (gets cropped on LinkedIn)

---

## Testing Your Design

Before posting:
1. ✅ View at actual size (100% zoom)
2. ✅ Check on mobile device (LinkedIn mobile app)
3. ✅ Test readability (can you read all text?)
4. ✅ Verify logo clarity (are they recognizable?)
5. ✅ Check color contrast (use online contrast checker)
6. ✅ Ensure file size under 5MB
7. ✅ Preview on LinkedIn (upload as draft first)
