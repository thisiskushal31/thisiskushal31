# Nexus - POS Platform | Purplle.com

## Project Overview

**Company:** Purplle.com  
**Project Type:** Production Platform - In-House POS Solution  
**Status:** Live & Operational  
**Platform:** Nexus - Point of Sale (POS) System  
**Deployment:** Deployed across retail stores  
**Role:** DevOps Engineer | Team Size: 5 people

## Executive Summary

Nexus is an in-house Point of Sale (POS) platform deployed across Purplle.com retail stores. The platform provides high-availability POS functionality, maintaining 99%+ uptime using GCP (GKE, GCR, Cloud SQL, Load Balancer, WAF, VPC). Successfully deployed and operational across retail store locations.

**Primary Goal:** Deploy high-availability Nexus (POS) system across retail stores with:
